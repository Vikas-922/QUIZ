## ❓ Simple Quiz Web App

A lightweight single-page quiz application where users can attempt multiple-choice questions and get instant results.

---

### 🚀 Features

- Single-page design
- Multiple-choice quiz with score display
- Stores quiz data and user scores in the database

---

### 🧑‍💻 Tech Stack

- **HTML, CSS, JavaScript** – Frontend
- **Express (Node.js)** – Backend
- **MongoDB** – Store quiz questions and scores

---

### ▶️ How It Works

1. Load the quiz on the main page.
2. Answer questions one by one.
3. See your final score at the end.
